# Student Course Management System

A Java console application developed for the COS201 Programming I Mid-Semester Lab Assessment.

## Features

- Add Course
- View All Courses
- Search Course by Code
- Compute Total Units
- Save Courses to File
- Load Courses from File
- Recursive Search
- Exception Handling

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- File I/O
- Exception Handling
- Recursion

## How to Run

Compile:

```bash
javac *.java
```

Run:

```bash
java Main
```
